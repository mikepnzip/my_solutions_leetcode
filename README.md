## LeetCode Stats

- **Languages Used**: Java (all solutions are now in Java)

## Repository Structure

- The solutions are organized into two main folders:
  - `C/`: Contains my initial solutions written in C.
  - `Java/`: Contains all 70 solutions rewritten in Java.
- Inside each folder, solutions are grouped into subfolders named like `problem_name123`, where `123` is the LeetCode problem number.
- Each subfolder contains the solution code.

## How to Navigate

1. Browse the `C/` or `Java/` directories to find solutions.
2. Look for subfolders like `problem_name123` to locate specific problems.
3. Feel free to explore and provide feedback!
